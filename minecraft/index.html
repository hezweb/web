Resource Packs:
Changes the appearance of your game and interface

  1. Start your Minecraft launcher and click PLAY
  2. At the Minecraft main menu, click Options... then click Resource Packs...
  3. Click Open Pack Folder
    • This opens the folder where you later need to place your resource packs, so keep it open for now!
  4. Download one of the following resource packs:
    1. The Excalibur 16x16 resource pack
      • Preview
      • No performance impact
      • Most players on this server use this resource pack
      • Use with Obscuria's Tools to easily distinguish between tools
    2. The Dokucraft Light 32x32 resource pack
      • Preview
      • Slight performance impact
    3. More resource packs can be found at Curseforge
  5. Place the downloaded resource pack file in your previously opened Pack Folder
    • The resource packs come as .zip files, do not unpack/extract them
  6. Go back to Minecraft and the new resource pack should be visible in the Available list
  7. Hover your mouse over the new resource pack and click the arrow icon to move the pack to the Selected list
  8. Click Done

OptiFine and shaders:
Better performance and extra graphical effects, like shadows, sunbeams and water reflections

Better performance:

  1. Make sure Minecraft isn't running, then download and install OptiFine
    • If you can't run the .jar file you'll have to install Java first
  2. Start your Minecraft launcher
  3. Make sure OptiFine is selected and click PLAY
  4. Once in Minecraft, click Options..., click Video Settings then click Performance...
  5. Switch every OFF option to ON, then click Done
    • You can leave Smooth FPS and Smooth World as OFF

Adding shaders:

  1. Click Shaders Folder
    • This opens the folder where you later need to place your shaders, so keep it open for now!
  2. Download one of the following shader packs:
    1. The BSL shader pack
      • Preview
      • Medium performance impact
      • Water is clearly visible in darker areas
      • Most players on this server use this shader pack
      • You can use my shader options, just place them in your shaders folder as a .txt
        Make sure the .txt has the same name as the pack file, for example BSL_v8.1.01.zip.txt
    2. Sildur's Vibrant shader pack
      • Preview
      • Has many versions depending on performance
      • Water is very hard to see in darker areas
    3. More shader packs can be found at the wiki
  3. Place the downloaded shader pack file in your previously opened Shaders Folder
    • The shader packs come as .zip files, do not unpack/extract them
  4. Go back to Minecraft and click the new shader pack visible in the Shaders list
  5. Click Done
  • Increase your FPS and decrease stuttering with performance tips

Reduce stuttering while moving:

  • Make sure OptiFine is installed and that OptiFine's performance settings are ON
    • Click Options then Video Settings
      • Click Performance...
        • Switch every OFF option to ON
        • You can leave Smooth FPS and Smooth World as OFF
  • Installing Minecraft to an SSD hard drive might help

Increasing FPS:
Press F3 in-game to see your FPS

  • Make sure OptiFine is installed and that OptiFine's performance settings are ON
    • Click Options then Video Settings
      • Click Performance...
        • Switch every OFF option to ON
        • You can leave Smooth FPS and Smooth World as OFF
  • Try lowering OptiFine's Render Quality
    • A shader or the shader called Internal must be selected in the Shaders list
    • Click Options, Video Settings then Shaders...
      • Right click Render Quality to lower it
      • This will make the image blurry
      • A Render Quality below 0.7x will be very blurry
  • Use a shorter Render Distance
    • Click Options then Video Settings
      • Lower Render Distance
      • This will lower how far you can see into the distance
      • A Render Distance below 4 is not recommended
      • A Render Distance above 12 will have no effect while playing on the server
  • Disable shaders
    • Click Options, Video Settings then Shaders...
      • Click OFF in the Shaders list
      • This will decrease image quality significantly

Features:

  • Password protected server
    • You will not be able to move or interact with the world until you've logged in
    • Press T on your keyboard and type /login <password> to login
      • Trusted players are not required to login
  • Skins for authenticated and unauthenticated players
    • Type /skins to select your skin
    • Type /skin url <skin.png url> [steve / slim] to upload your own skin
      • You can use skin urls directly from minecraftskins.com by clicking Image Link and copying the link
  • You can view the world map online by going here
    • Will be unavailable during maintenance
    • Does not render caves with no natural sky light and never below depth 0
    • Type /bluemap marker create <id> <map-id> [x y z] <label> to place a marker on the world map
      • Accepted map-ids are world, nether and end
      • You can get x y and z coordinates from the online world map, or leave it empty to use your curren in-game coordinates
    • Type /bluemap marker remove <id> to remove a marker
    • Type /bluemap marker list to list all existing markers and their ids
  • The last 7 world backups can be found here
    • May be unavailable during maintenance
  • Check server latency/ping
    • Type /ping to see your own ping
      • A ping below 115ms is usually fine
    • Type /ping <player> to see another players ping
  • Display where mobs can spawn
    • Type /LL to toggle mob spawn indication
      • Smoke indicates a block dark enough for mobs to spawn on
  • Sort container items
    • Right click outside any container (inventory, chest etc.) to automatically sort it
      • Only works once every second
    • Type /sortingconfig categories set <category> if you want to change sorted categories
    • Type /sortingconfig categories reset to reset to default categories
  • Refill hotbar items
    • All item types enabled by default
    • Type /sortingconfig refill <BLOCKS/BREAKABLES/CONSUMABLES> <true/false> to set refill for a specific item type
    • Type /sortingconfig refill <true/false> to set refill for all item types
  • Beehive information
    • Right click a beehive to get in-game information about it
    • Only works when your hands are empty
  • Playtime
    • Type /playtime to see your total play time
    • Type /playtime <player> to see another players total play time
  • Death compass
    • A death compass is given to any player that dies
    • The death compass always points to the last location where the player died
    • Dropping your current death compass will destroy it forever
  • Away from keyboard
    • A player enters AFK mode after 3 minutes of standing still
    • Moving will force the player out of AFK mode
    • A player in AFK mode will not block sleeping attempts
    • A [AFK] suffix is applied to AFK players in the TAB player list
    • Type /afk to manually enter or leave AFK mode
    • Type /afk <reason> to enter AFK mode with a broadcasted reason
  • Private messages
    • Type /pm <player> <your message> to send a private message to any player
    • Type /r <your message> reply to the last private message you
  • Mail
    • Send messages to offline players by using the mail system
    • Players are notified about received mail during login
    • Type /mail send <player> <your message> to send mail to any player
    • Type /mail read to read your received mail
    • Type /mail for more mail commands
  • Hats!
    • Most items can be placed in your head equipment slot
  • Me Action
    • Type /me <does something> to display a IRC styled me action message
  • Recipes
    • Type /recipe <item_name> to display the recipe for that item
      • Typing /formula <item_name> also works
    • Type /itemdb <item_name> to search for an item name
